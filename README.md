# AI Resume Analyzer & Resume Optimizer

An AI-powered resume analysis platform that compares resumes against job descriptions, identifies skill gaps, evaluates ATS compatibility, generates recommendations, and creates optimized job-specific resume versions.

## Features

### Resume Analysis
- Upload PDF, DOCX, or TXT resumes
- Extract and process resume content
- Analyze resume against job descriptions
- ATS compatibility scoring

### Skill Gap Analysis
- Detect matching skills
- Identify missing skills
- Calculate resume-job match percentage
- Generate improvement recommendations

### AI Feedback
- AWS Bedrock integration
- AI-powered resume evaluation
- Interview readiness assessment
- Personalized improvement suggestions

### Resume Optimization
- Generate tailored resumes for target jobs
- Improve professional summaries
- Optimize experience bullet points
- Export polished DOCX resumes

## Technologies Used

- Python
- Streamlit
- AWS Bedrock
- Boto3
- PDFPlumber
- Python-Docx
- Natural Language Processing (NLP)
