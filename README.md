# AI Resume Analyzer

AI-powered resume analyzer built with Python, Streamlit, AWS Bedrock, and NLP techniques.

## Features

- Upload PDF and DOCX resumes
- Paste job descriptions
- Calculate resume-job match score
- Detect missing skills
- Generate recommendations
- Interview readiness scoring
- AI-powered feedback using AWS Bedrock

## Tech Stack

- Python
- Streamlit
- AWS Bedrock
- Boto3
- PDFPlumber
- Python-Docx

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Example Output

- Match Score
- Matched Skills
- Missing Skills
- Recommendations
- Interview Readiness Score
- AWS Bedrock Feedback

## Project Goal

Help job seekers quickly identify skill gaps between their resume and a target job description using AI-powered analysis.
